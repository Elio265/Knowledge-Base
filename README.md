## 使用方式

1. Obsidian 打开 `obsidian-cpp-kb/` 作为 Vault
2. 从 `00-MOC/C++知识地图.md` 开始
3. 利用 `[[双向链接]]` 在知识之间跳转

## 知识状态

| status | 含义 |
|--------|------|
| 待学习 | 标记了但还没开始 |
| 学习中 | 知道概念，缺少实践或细节 |
| 已掌握 | 可以轻松讲解、手写代码 |
| 深入理解 | 理解底层原理和设计权衡 |
| 面试重点 | 高频考点，需要反复复习 |

## AI 知识库管理规则

维护规则在隐藏配置文件 `.claude/rules.md` 中，不作为 Obsidian 笔记跳转。核心原则：

- **先更新 MOC，再新增笔记**
- **一个知识点 = 一个文件**
- **更新优于新增，链接优于孤立**
