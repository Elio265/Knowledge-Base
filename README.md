## 使用方式

1. 用 Obsidian 打开 `obsidian-cpp-kb/` 目录作为 Vault
2. 从 `C++知识地图.md` 开始浏览
3. 利用双向链接 `[[]]` 在知识之间跳转
4. 随着学习深入，持续更新各文件的 `status` 标签

## 状态标记说明

| 标记 | 含义 |
|------|------|
| ✅ 已掌握 | 可以轻松讲解、手写代码 |
| 📖 需要补充 | 知道概念但缺少实践或细节 |
| 🔬 深入学习 | 需要进一步研究的高级主题 |

## 维护计划

- 每学完一个主题，更新对应文件的状态和内容
- 遇到新的面试题或坑，添加到对应文件的对应章节
- 定期回顾知识地图，检查知识体系的完整性
